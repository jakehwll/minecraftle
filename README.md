# Minecraftle

A daily puzzle game fusing Wordle and Minecraft crafting recipes built with Next.js.

[Play Minecraftle](https://minecraftle.zachmanson.com)

## Setting Up

1. Clone the repo
2. `bun install`
3. `bun run dev`
